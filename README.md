# Website Penjualan Produk Anime

Website sederhana yang dibuat untuk menampilkan dan menjual berbagai produk bertema anime seperti figure, manga, merchandise, novel, dan plush.

## Deskripsi

Website ini menampilkan halaman utama dengan beberapa kategori produk yang dapat dipilih oleh pengguna. Pengguna juga dapat mencari produk melalui fitur pencarian yang tersedia di halaman utama.

## Fitur

* Halaman Home
* Pencarian produk
* Kategori produk
* Tampilan gambar produk

## Teknologi yang Digunakan

* HTML
* CSS

## Tujuan Project

Project ini dibuat sebagai tugas **Ujian Tengah Semester (UTS)** semasa saya kuliah untuk mempelajari dasar-dasar pembuatan website.

## Cara Menjalankan

1. Download atau clone repository ini.
2. Buka folder project.
3. Jalankan file **home.html** di browser.
